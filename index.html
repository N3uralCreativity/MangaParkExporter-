üöÄ How It Works

Get started in 4 simple steps

1

Download & Launch

Download the standalone .exe (no Python required) or run from source. The application launches in fullscreen with an intuitive GUI.

2

Choose Mode & Configure

Select Authenticated mode (with MangaPark cookies) or Public mode (no login). Follow the tutorial to extract cookies from DevTools if needed.

3

Start Export Process

Click "Start Export" and watch real-time progress. The app scrapes MangaPark, enriches with MAL IDs using Jikan API, and shows detailed logs.

4

View & Import Results

Open the generated HTML to browse your collection or import the XML file directly to MyAnimeList. All files saved in output/ folder.
